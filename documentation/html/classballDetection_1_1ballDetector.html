Detailed Description

Class which define the color detection algorithm used which run in a node on his own.

Receive in input compressed images from the camera and then apply a mask of different colors to detect if at least one colored ball is in the scene. It's important to highlight that colors can be detected only once, so after the first detection of the black ball the algorithm is not able to detect other black balls.
