def knowledgeRep.Rooms.explore (   self)

Explore function that returns a random position away from the rooms already visited and the position reached during the FIND mode.

Basically it generate a random position and check if it belongs in the neighborhood of each detected room (or position), if so it will reach such location. See the README for more details.

Returns a array which contains a neighborhood of a given number.

For instance if a = 1 and l = 2 it'll return [-1, 0, 1, 2, 3]. This method guaranties, that given a point (x,y), to return a 2lX2l area around the point.
