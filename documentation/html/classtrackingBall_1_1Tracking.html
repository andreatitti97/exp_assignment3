Detailed Description

Class used for tracking the balls detected by ballDetect.py.

After receiving the request from the client (Cmd_Manager_Node) the script start to track the ball of the color sended by the client. It also save the robot position subscribing to /odom topic for store the room position. Sice during the tracking the move_base action is aborted, it's also implemented a simple obstacle avoidance using the "bug algorithm" for prevent the robot collide with obstacle during the approaching to the ball.
